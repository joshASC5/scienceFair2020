What is Friction?

Friction is a force that resists motion caused by some sort of force between two objects. Friction is present in all aspects of life, from sliding a box across a floor to an object in free‑fall. Friction is always tangential to the contact surface, and it only acts when another force is present that tries to move an object.

There are two types of friction: static friction (denoted as ƒs) and kinetic friction (denoted as ƒk). An object is said to experience static friction when the object remains at rest when a force is acting on the object that has a nonzero component parallel to the ground. In other words, if I were to push a heavy box that does not move, the box is experiencing static friction that opposes my push (i.e., Newton's Third Law). If the box starts moving, that indicates that my push was stronger than the maximum value of static friction and the box is now experiencing kinetic friction (more on that later). Kinetic friction is generally weaker than static friction, which explains why it is easier to push or pull an already-moving object than it is to move an object from rest.
